Key Technologies

Project Description

The new website was designed by an outside agency, but largely developed by me in-house to reduce costs associated with the project. Frontend frameworks were dispensed with to allow for a more design flexibility. Only a few external libraries were used throughout the site.

The frontend assets were built in SCSS, with the Block Element Modifier methodology. In addition to the standard HTML elements, there are about 20 design elements used in multiple ways throughout the site.

The site is built within ExpressionEngine, which features a web IDE, but all changes are made locally, packaged, and deployed through Git.

Blue Water Task Force

Progressive Web App to display recreational water quality data

calendar_today Jan, 2020

Key Technologies

Javascript GraphQL d3.js HTML Progressive Web App Web Components AWS Amplify NoSQL

Project Description

This was a full frontend and backend project. The previous website had not been modified in over 5 years, so this was a full tear-down-and-rebuild.

The look of the platform based around the brand aesthetic at the time. It makes heavy use of Web Components with a custom CSS theme. One of the goals was to make it useable from a mobile device, and the platform is able to be installed as a Progressive Web App

The backend environment is powered by AWS DynamoDB (NoSQL), with an AWS Appsync API (GraphQL). It is a full CRUD application for multiple models / datatypes.
